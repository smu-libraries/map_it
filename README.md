# map_it

map_it provides the location map for a physical item in the Libraries' catalog. It is meant for use with Ex Libris Primo/Alma (see relevant [Knowledge Article](http://knowledge.exlibrisgroup.com/Alma/Knowledge_Articles/How_to_configure_Alma_to_display_a_map_to_a_physical_item_in_Primo).)
